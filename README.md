# Simple Text to Speech

A lightweight web application that converts text into speech using the **Google Gemini API**.  
Built with **React, TypeScript, and Vite**, featuring real-time audio visualization, playback controls, and audio download support.

---

##  Features

- Convert text to natural-sounding speech
- Play / top audio output
- Download generated audio
- Clean, modular React component structure

---

## Tech Stack

- **Frontend:** React + TypeScript
- **Build Tool:** Vite
- **API:** Google Gemini Text-to-Speech
- **Audio Handling:** Web Audio API
- **Styling:** CSS (custom)
